# BoxPeer website file

## Installation
1. **Clone the repo**
    ```bash
       git clone https://github.com/Priceless-P/BoxPeer-Web.git
       cd ./BoxPeer-Web
    ```
2. **Set env**
    ```bash
     cp .env-sample .env
    ```
2. **Install the dependencies:**

   ```bash
   npm install
   ```

2. **Start the backend app:**

   ```bash
   cd ..
   cargo run
   ```
